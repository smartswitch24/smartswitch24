## After Generation – Save & Convert

Once you have the images (PNG or JPG from the AI tool), convert and save as WebP using one of:

```bash
# cwebp (Google) – convert single file
cwebp -q 85 input.png -o 01-hurghada-underwater-sea-turtle.webp

# ffmpeg – batch convert all PNG in folder
for f in *.png; do ffmpeg -i "$f" -quality 85 "${f%.png}.webp"; done

# ImageMagick – batch convert
mogrify -format webp -quality 85 *.png
```

Target file size: **150–400 KB per image** at 1600×900.
